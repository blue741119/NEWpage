Project 02・Canvas 動畫

霓虹彈跳球

從一顆靜止的圓開始，逐步加入移動、反彈、擋板、目標、計分與霓虹效果，最後組成完整遊戲。

先把遊戲拆成小零件

Canvas顯示背景、球、擋板與目標。

步驟 1

建立響應式 Canvas

Canvas 的 width 與 height 是內部繪圖座標；CSS 的寬度則控制它在螢幕上的顯示大小。

<canvas id="myCanvas" width="960" height="600"></canvas>
<script src="content.js"></script>
canvas {
  display: block;
  width: 100%;
  height: auto;
  background: #080c1e;
}

這樣桌機仍使用 960 × 600 的清楚座標，手機則會等比例縮小，不會產生水平捲動。

步驟 2

先畫出一顆球

球需要 X、Y 座標、半徑與水平／垂直速度。使用 arc() 畫圓。

const ball = {
  x: canvas.width / 2,
  y: 430,
  radius: 15,
  vx: 285,
  vy: -310
};

ctx.beginPath();
ctx.arc(ball.x, ball.y, ball.radius, 0, Math.PI * 2);
ctx.fillStyle = '#63e6ff';
ctx.fill();

vx 是每秒水平方向移動量，vy 是垂直方向移動量；負的 Y 代表向上。

步驟 3

用動畫迴圈讓球移動

requestAnimationFrame 會配合螢幕更新頻率執行。利用兩幀的時間差，讓不同電腦上的球速保持一致。

let lastTime = performance.now();

function gameLoop(time) {
  const delta = Math.min((time - lastTime) / 1000, 0.032);
  lastTime = time;

  ball.x += ball.vx * delta;
  ball.y += ball.vy * delta;
  draw();
  requestAnimationFrame(gameLoop);
}

requestAnimationFrame(gameLoop);

每一幀都要先清除或重畫背景，否則球移動時會留下永久的軌跡。

步驟 4

碰到邊界就反彈

球心不能超過「畫布尺寸減半徑」。碰到左右邊界時反轉 vx，碰到上下邊界時反轉 vy。

if (ball.x + ball.radius >= canvas.width) {
  ball.x = canvas.width - ball.radius;
  ball.vx = -Math.abs(ball.vx);
} else if (ball.x - ball.radius <= 0) {
  ball.x = ball.radius;
  ball.vx = Math.abs(ball.vx);
}

if (ball.y - ball.radius <= 0) {
  ball.y = ball.radius;
  ball.vy = Math.abs(ball.vy);
}

先把球移回合法位置，再改變速度，可避免球卡在邊界上連續反轉。

步驟 5

加入滑鼠與觸控擋板

由於 Canvas 會隨螢幕縮放，必須把畫面上的指標座標換算成 Canvas 內部座標。

function movePaddle(clientX) {
  const rect = canvas.getBoundingClientRect();
  const canvasX = (clientX - rect.left) *
    (canvas.width / rect.width);

  paddle.x = Math.max(0, Math.min(
    canvas.width - paddle.width,
    canvasX - paddle.width / 2
  ));
}

canvas.addEventListener('pointermove', event => {
  movePaddle(event.clientX);
});

pointermove 同時支援滑鼠、觸控筆與手指，比只使用 mousemove 更完整。

步驟 6

建立目標方塊與碰撞

每個方塊保存位置、尺寸與是否仍可見。用最接近圓心的矩形座標，判斷圓與矩形是否接觸。

function circleTouchesRect(circle, rect) {
  const closestX = Math.max(
    rect.x,
    Math.min(circle.x, rect.x + rect.width)
  );
  const closestY = Math.max(
    rect.y,
    Math.min(circle.y, rect.y + rect.height)
  );
  const dx = circle.x - closestX;
  const dy = circle.y - closestY;
  return dx * dx + dy * dy <=
    circle.radius * circle.radius;
}
if (circleTouchesRect(ball, brick)) {
  brick.visible = false;
  score += 100;
  ball.vy *= -1;
}

正式版本還會比較球的上一個位置，判斷它從側面還是上下方撞擊，決定要反轉哪一個速度。

步驟 7

加入立體球、拖尾與霓虹效果

徑向漸層讓球看起來具有高光和深度，陰影則產生能量光暈。

const glow = ctx.createRadialGradient(
  ball.x - 6, ball.y - 7, 2,
  ball.x, ball.y, ball.radius
);
glow.addColorStop(0, '#ffffff');
glow.addColorStop(0.25, '#baf7ff');
glow.addColorStop(0.6, '#55dff5');
glow.addColorStop(1, '#4872de');

ctx.shadowColor = '#5deaff';
ctx.shadowBlur = 22;
ctx.fillStyle = glow;

拖尾是一組較舊的球座標。越舊的座標畫得越小、越透明，就能形成速度感。

trail.unshift({ x: ball.x, y: ball.y });
if (trail.length > 13) trail.pop();

trail.forEach((point, index) => {
  const opacity = 1 - index / trail.length;
  ctx.globalAlpha = opacity * 0.2;
  // 在 point.x、point.y 畫出較小的圓
});
