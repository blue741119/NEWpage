Project 04・Canvas 生存遊戲

殭屍包圍戰

利用鍵盤控制、直線與曲線軌跡、碰撞判定、生命值和瀏覽器儲存空間，完成節奏快速的生存遊戲。

開始之前

把生存遊戲拆成系統

人物控制使用方向鍵組合水平與垂直移動。
移動軌跡物件隨機選擇直線或曲線穿越場地。
生命與補血碰撞扣血，醫療包回復生命。
本機排名把名字和分數存在 localStorage。

步驟 1

建立響應式遊戲畫面

Canvas 使用固定的 960 × 600 內部座標，CSS 再依螢幕寬度縮放，因此碰撞計算不會受到手機尺寸影響。

<div class="health-track">
  <div id="health-fill" class="health-fill"></div>
</div>
<strong id="health-value">100</strong>

<canvas id="game" width="960" height="600"></canvas>
canvas {
  display: block;
  width: 100%;
  height: auto;
  touch-action: none;
}

Canvas 保持固定內部座標，CSS 只負責縮放顯示，因此不同螢幕上的移動與碰撞規則仍然一致。

步驟 2

使用鍵盤方向鍵控制人物

按下方向鍵時把按鍵加入集合，放開時移除。這樣可以同時按住兩個方向鍵進行斜向移動。

const pressedKeys = new Set();

window.addEventListener('keydown', event => {
  if (event.key.startsWith('Arrow')) {
    event.preventDefault();
    pressedKeys.add(event.key);
  }
});

window.addEventListener('keyup', event => {
  pressedKeys.delete(event.key);
});

斜向移動前先把向量正規化，避免同時按兩個方向時速度變快。

let x = Number(pressedKeys.has('ArrowRight'))
  - Number(pressedKeys.has('ArrowLeft'));
let y = Number(pressedKeys.has('ArrowDown'))
  - Number(pressedKeys.has('ArrowUp'));

const length = Math.hypot(x, y) || 1;
player.x += (x / length) * player.speed * delta;
player.y += (y / length) * player.speed * delta;
步驟 3

從四面八方高速穿越

先隨機選擇上、右、下、左其中一邊，設定朝畫面內的基準角度，再加入少量隨機偏移。

const edge = Math.floor(Math.random() * 4);
const startData = [
  { x: Math.random() * width, y: -20, angle: Math.PI / 2 },
  { x: width + 20, y: Math.random() * height, angle: Math.PI },
  { x: Math.random() * width, y: height + 20, angle: -Math.PI / 2 },
  { x: -20, y: Math.random() * height, angle: 0 }
][edge];

殭屍速度提高為約 125～235，生存時間越長還會額外加速。每過 20 秒，每一波生成數量翻倍；產生前會計算剩餘名額，讓場上普通殭屍最多保持在 100 隻。

const waveSize = 2 ** Math.floor(elapsed / 20);
const spawnCount = Math.min(waveSize, 100 - zombies.length);
for (let i = 0; i < spawnCount; i += 1) spawnZombie();

const interval = Math.max(.17, .59 - elapsed * .004);
const speedBoost = Math.min(elapsed * 0.9, 80);
const speed = 125 + Math.random() * 110 + speedBoost;
步驟 4

固定走直線或曲線

殭屍出生後不再追蹤人物。直線的轉向量為 0；曲線則每一幀慢慢改變角度。

const curved = Math.random() < 0.5;
zombie.curve = curved
  ? (Math.random() < .5 ? -1 : 1) *
    (.16 + Math.random() * .28)
  : 0;

zombie.angle += zombie.curve * delta;
zombie.x += Math.cos(zombie.angle) * zombie.speed * delta;
zombie.y += Math.sin(zombie.angle) * zombie.speed * delta;

殭屍移出畫面邊緣後立即從陣列刪除，避免看不見的物件繼續佔用效能。

const outside = zombie.x < -margin ||
  zombie.x > width + margin ||
  zombie.y < -margin ||
  zombie.y > height + margin;

if (outside) zombies.splice(index, 1);
步驟 5

碰撞扣除 10 點生命

人物和殭屍都視為圓形。兩個圓心的距離小於半徑總和，就代表碰撞。

function touching(a, b) {
  return Math.hypot(a.x - b.x, a.y - b.y)
    <= a.radius + b.radius;
}

if (touching(zombie, player)) {
  player.health = Math.max(0, player.health - 10);
}

血量從 100 開始，畫面以百分比控制血條寬度；歸零時停止更新遊戲並開啟姓名輸入對話框。

healthFill.style.width = `${player.health}%`;

if (player.health === 0) {
  running = false;
  showGameOver();
}
步驟 6

每分鐘出現 5 個醫療包

每 12 秒產生一個醫療包，因此一分鐘最多出現 5 次。每個醫療包會隨機選擇快／慢速度以及直線／曲線軌跡。

let medkitSpawnTimer = 12;

if (medkitSpawnTimer <= 0) {
  spawnMedkit();
  medkitSpawnTimer = 12;
}

const fast = Math.random() < .5;
const speed = fast
  ? 130 + Math.random() * 50
  : 68 + Math.random() * 30;

醫療包移出邊緣時同樣刪除；人物碰到它則回復 20 點，且不會超過 100。

if (touching(medkit, player)) {
  player.health = Math.min(100, player.health + 20);
  bonusScore += 50;
} else if (leftPlayArea(medkit)) {
  medkits.splice(index, 1);
}
步驟 7

加入彩虹鱗片龍頭

龍頭每 10 秒出現在下一個角落，在場上停留 30 秒。它不移動，但每 5 秒朝場地內的隨機位置投射唾液。造型參考彩虹毒龍圖，加入大型彎角、多色菱形鱗片、發光龍眼、臉側尖刺與獠牙。

let mutantSpawnTimer = 10;

if (mutantSpawnTimer <= 0) {
  spawnMutant();
  mutantSpawnTimer += 10;
}

const mutant = {
  x: corner.x,
  y: corner.y,
  life: 30,
  spitTimer: 5
};

const targetX = 55 + Math.random() * (width - 110);
const targetY = 55 + Math.random() * (height - 110);

唾液不再瞄準玩家，而是隨機落在場地內。毒液會繼承龍頭顏色；落地後建立同色、半徑 87 的污染區，也就是原本半徑的 3 倍，並存在 5 秒。玩家站在裡面每秒扣 2 點，每一灘最多扣 10 點。

const puddle = {
  x: targetX,
  y: targetY,
  radius: 87,
  color: spit.color,
  life: 5,
  damageTimer: 0,
  totalDamage: 0
};

if (touchingPuddle && puddle.totalDamage < 10) {
  puddle.damageTimer += delta;
  if (puddle.damageTimer >= 1) {
    player.health -= 2;
    puddle.totalDamage += 2;
    puddle.damageTimer -= 1;
  }
}
步驟 8

輸入名字並建立排行榜

遊戲結束後把名字、分數和生存時間加入陣列，依分數排序，只保存前 10 名。

const entries = JSON.parse(
  localStorage.getItem('zombieSurvivorRanking')
) || [];

entries.push({ name, score, time });
entries.sort((a, b) => b.score - a.score);

localStorage.setItem(
  'zombieSurvivorRanking',
  JSON.stringify(entries.slice(0, 10))
);

排行榜只保存在目前的瀏覽器裝置。若要讓所有玩家共用排名，下一步可以串接 Supabase 資料庫。

步驟 9

組合成完整生存遊戲
