Lesson 03・互動

JavaScript 入門

JavaScript 像房子的電力系統：它讓按鈕、表單與畫面能對使用者的操作做出反應。

1. 把 JavaScript 連到 HTML

建立 script.js，並在 body 結束標籤前載入它。

<script src="script.js"></script>

2. 變數與函式

變數保存資料，函式則把一組動作包起來，之後可以重複呼叫。

const name = '小林';

function greet() {
  return `你好，${name}！`;
}

3. 回應點擊事件

先選取 HTML 元素，再用 addEventListener 監聽點擊。

const button = document.querySelector('#hello-button');
const result = document.querySelector('#hello-result');

button.addEventListener('click', () => {
  result.textContent = '成功了！';
});

4. 動手練習

任務：在個人名片加入「顯示聯絡方式」按鈕。第一次點擊顯示 Email，再次點擊則隱藏。
