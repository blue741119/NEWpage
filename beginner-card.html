步驟 1

建立資訊列與卡牌區

HTML 只準備容器；卡牌會依關卡由 JavaScript 動態產生。

<div class="hud">
  <strong id="level">1 / 5</strong>
  <strong id="score">0</strong>
  <strong id="moves">0</strong>
</div>

<section id="board" class="board"></section>
<button id="restart">重新開始</button>
步驟 2

設定五個關卡

陣列中的每個數字就是該關的卡牌總數。全部都是偶數，才能兩兩配對。

const levelCounts = [6, 8, 10, 12, 14];
let currentLevel = 0;

const cardCount = levelCounts[currentLevel];
const pairCount = cardCount / 2;

第一關有 3 組圖案，第五關則有 7 組圖案。

步驟 3

製作成對牌組並洗牌

先從圖案庫取出需要的組數，每個圖案複製兩張，最後使用 Fisher–Yates 演算法洗牌。

const pairs = symbols.slice(0, pairCount).flatMap(
  (symbol, pairIndex) => [
    { ...symbol, pairIndex },
    { ...symbol, pairIndex }
  ]
);
function shuffle(items) {
  const result = [...items];
  for (let i = result.length - 1; i > 0; i -= 1) {
    const random = Math.floor(Math.random() * (i + 1));
    [result[i], result[random]] = [result[random], result[i]];
  }
  return result;
}
步驟 4

把牌組轉成按鈕

每張卡牌都是可用鍵盤操作的 button，配對編號放在 data-pair。

function createCard(data) {
  const card = document.createElement('button');
  card.type = 'button';
  card.className = 'card';
  card.dataset.pair = data.pairIndex;
  card.innerHTML = `
    <span class="card-back"></span>
    <span class="card-front">${data.icon}</span>`;
  return card;
}
步驟 5

翻兩張後比較圖案

第一張先暫存；第二張翻開後比較兩張的配對編號。比較期間用鎖定狀態阻止使用者繼續翻牌。

let firstCard = null;
let secondCard = null;
let boardLocked = false;

function flipCard(card) {
  if (boardLocked || card === firstCard) return;
  card.classList.add('flipped');

  if (!firstCard) {
    firstCard = card;
    return;
  }

  secondCard = card;
  boardLocked = true;
  const matched = firstCard.dataset.pair ===
    secondCard.dataset.pair;
}

如果不同，等待約 850 毫秒再蓋回去，讓玩家有時間看清第二張圖案。

步驟 6

配對成功、得分與過關

配對成功後保留翻開、停用按鈕並加分。找到的組數等於本關總組數時顯示過關畫面。

matchedPairs += 1;
score += 100 + currentLevel * 25;
firstCard.disabled = true;
secondCard.disabled = true;

if (matchedPairs === cardCount / 2) {
  showLevelResult();
}
function advanceLevel() {
  currentLevel += 1;
  startLevel();
}

關卡越高，每組配對的分數也越高；完成第五關後可重新挑戰第一關。

步驟 7

用 CSS 製作翻牌動畫

卡牌內層保留 3D 空間，正面先轉到背後；加入 flipped 後整張旋轉 180 度。

.card-inner {
  transform-style: preserve-3d;
  transition: transform .48s ease;
}

.card-front {
  transform: rotateY(180deg);
}

.card.flipped .card-inner,
.card.matched .card-inner {
  transform: rotateY(180deg);
}

卡片使用彈性 Grid，在桌機、平板和手機上會自動調整欄數與尺寸。

步驟 8

累積遊戲時間並建立前 10 名

玩家第一次翻牌才開始計時；完成一關後暫停，下一關第一次翻牌再繼續。五關的實際遊玩時間會累積在一起。

let totalElapsedMs = 0;
let levelStartedAt = null;

function stopLevelTimer() {
  totalElapsedMs += performance.now() - levelStartedAt;
  levelStartedAt = null;
}

完成第五關後輸入名字，依完成時間由少到多排序，只保存前 10 名。

entries.push({ name, timeMs: totalElapsedMs });
entries.sort((a, b) => a.timeMs - b.timeMs);

localStorage.setItem(
  'cardMemoryTimeRanking',
  JSON.stringify(entries.slice(0, 10))
);

完整成品還包含關卡進度、總分、翻牌次數、即時計時、配對提示與重新開始功能。下方可直接遊玩。
