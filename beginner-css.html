CSS 入門

CSS 像房子的裝潢：它控制顏色、字體、空間，以及內容如何排列。

1. 把 CSS 連到 HTML

建立 style.css，再把下面這行放進 HTML 的 head。

<link rel="stylesheet" href="style.css">

2. 選擇器與屬性

選擇器指出要改誰，大括號裡則寫要改成什麼樣子。

h1 {
  color: #5b4ae8;
  font-size: 40px;
}

.card {
  background: white;
  border-radius: 16px;
}

h1 選取所有主標題；前面有句點的 .card 選取 class 為 card 的元素。

3. 理解盒模型

每個元素都是一個盒子：內容外面依序是內距 padding、邊框 border、外距 margin。

.card {
  padding: 24px;
  border: 1px solid #dfe4ef;
  margin: 20px 0;
}

4. 用 Grid 排版

Grid 很適合排列卡片。螢幕夠寬時顯示三欄，較窄時可再用媒體查詢改成一欄。

HTMLCSSJS
.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
