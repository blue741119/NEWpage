Project 01・完整實作

森林貪食蛇

把 HTML、CSS 與 JavaScript 組合成一款完整遊戲。每一步先理解目的，再加入一小段程式碼。

直接玩完整成品從第一步開始
開始之前

先拆解遊戲需要什麼

Canvas負責畫出遊戲場地、蛇與蘋果。
陣列依序保存蛇頭到蛇尾的位置。
遊戲迴圈每隔固定時間更新一次位置。
事件監聽接收方向鍵、WASD 和空白鍵。

專案由兩個檔案組成：

Snake/ ├── index.html　　畫面、計分板與樣式 └── content.js　 遊戲規則與 Canvas 繪圖
步驟 1

建立遊戲畫布

HTML 的 canvas 是一塊可由 JavaScript 繪圖的區域。先建立 320 × 320 像素畫布、分數文字與重新開始按鈕。

<p id="myScore">遊戲分數：0</p>
<p id="myHighestScore">最高分數：0</p>
<canvas id="myCanvas" width="320" height="320"></canvas>
<button id="Start" type="button">開始新遊戲</button>
<script src="content.js"></script>

在 JavaScript 取得畫布和 2D 繪圖工具：

const canvas = document.querySelector('#myCanvas');
const ctx = canvas.getContext('2d');
const unit = 20;

unit 是每次移動的距離。320 ÷ 20 = 16，因此場地共有 16 × 16 格。

步驟 2

用陣列表示一條蛇

每個物件代表蛇身的一節，第一個物件永遠是蛇頭。所有座標都保持 20 的倍數，蛇才會對齊格線。

let snake = [
  { x: 80, y: 40 }, // 蛇頭
  { x: 60, y: 40 },
  { x: 40, y: 40 },
  { x: 20, y: 40 }  // 蛇尾
];

let direction = 'Right';

移動的核心概念是：在陣列最前面加入新蛇頭，再移除最後一節蛇尾。蛇的長度不變，但看起來就向前移動了。

步驟 3

建立遊戲迴圈，讓蛇前進

先根據方向計算新座標，再用餘數讓蛇從右邊離開時回到左邊。

function tick() {
  const head = snake[0];
  const newHead = {
    x: (head.x + 20 + canvas.width) % canvas.width,
    y: head.y
  };

  snake.unshift(newHead);
  snake.pop();
  drawScene();
}

const timer = setInterval(tick, 115);

setInterval 每 115 毫秒執行一次 tick。數字越小，蛇移動越快。

步驟 4

加入四方向鍵盤控制

使用方向向量，把 Right、Down、Left、Up 分別轉換成 X、Y 的變化量。

const directions = {
  Right: { x: 1, y: 0 },
  Down:  { x: 0, y: 1 },
  Left:  { x: -1, y: 0 },
  Up:    { x: 0, y: -1 }
};

window.addEventListener('keydown', event => {
  const keyMap = {
    ArrowRight: 'Right', ArrowDown: 'Down',
    ArrowLeft: 'Left', ArrowUp: 'Up'
  };
  if (keyMap[event.key]) direction = keyMap[event.key];
});

正式版本還會檢查反方向：向右移動時不能立刻轉左，避免蛇頭直接撞進身體。

步驟 5

產生果實，吃到後讓蛇成長

隨機座標也必須對齊格線，並確認果實沒有出現在蛇身上。

function pickFruitLocation() {
  let position;
  do {
    position = {
      x: Math.floor(Math.random() * 16) * unit,
      y: Math.floor(Math.random() * 16) * unit
    };
  } while (snake.some(part =>
    part.x === position.x && part.y === position.y
  ));
  return position;
}

當新蛇頭與果實座標相同時，不要移除蛇尾，蛇就會多出一節；同時增加分數並重新選擇果實位置。

if (newHead.x === fruit.x && newHead.y === fruit.y) {
  score += 1;
  fruit = pickFruitLocation();
} else {
  snake.pop();
}
步驟 6

判斷蛇是否撞到自己

在加入新蛇頭前，用 some() 檢查新座標是否與身體任一節相同。

const hitBody = snake.some(part =>
  part.x === newHead.x && part.y === newHead.y
);

if (hitBody) {
  clearInterval(timer);
  gameOver = true;
  drawScene();
  return;
}

若這一步放在移動之後才檢查，畫面可能會多走一格；遊戲規則的執行順序非常重要。

步驟 7

把方塊畫成蛇與蘋果

蛇身用圓角線連接每節中心，再用圓形與漸層增加立體感。蛇頭另外畫橢圓、眼睛和舌頭。

ctx.lineCap = 'round';
ctx.lineJoin = 'round';
ctx.lineWidth = 14;
ctx.strokeStyle = '#53ad68';

ctx.beginPath();
ctx.moveTo(bodyX, bodyY);
ctx.lineTo(nextX, nextY);
ctx.stroke();

蘋果則用三個相疊的圓形形成果肉，再補上果梗、葉片和高光：

ctx.fillStyle = '#ef3f48';
ctx.beginPath();
ctx.arc(cx - 4, cy, 7, 0, Math.PI * 2);
ctx.arc(cx + 4, cy, 7, 0, Math.PI * 2);
ctx.arc(cx, cy + 4, 7, 0, Math.PI * 2);
ctx.fill();
步驟 8

組合成完整遊戲

最後加入最高分的 localStorage、空白鍵暫停、遊戲結束遮罩與重新開始按鈕。下方嵌入的就是 Snake/index.html 成品，可以直接操作。
