Lesson 01・結構

HTML 入門

HTML 像房子的骨架：它告訴瀏覽器哪些內容是標題、段落、圖片與連結。

1. 建立頁面骨架

每個 HTML 文件都從文件類型開始，內容分成 head 與 body。

<!doctype html>
<html lang="zh-Hant-TW">
<head>
  <meta charset="utf-8">
  <title>我的第一個網頁</title>
</head>
<body>
  網頁上看得到的內容放在這裡
</body>
</html>

大部分標籤成對出現，例如 <p>內容</p>；前面是開始標籤，後面是結束標籤。

2. 標題與段落

h1 是頁面最重要的標題，p 代表一個段落。

小林的作品集

我正在學習製作網頁，這是我的第一個作品。

<h1>小林的作品集</h1>
<p>我正在學習製作網頁，這是我的第一個作品。</p>

3. 圖片與連結

圖片使用 img，連結使用 a。替圖片寫清楚的 alt，讓無法看到圖片的人也能理解內容。

<img src="photo.jpg" alt="小林在書桌前寫程式">
<a href="https://example.com">查看我的作品</a>

4. 動手練習

任務：建立一張個人名片，加入一個主標題、一段自我介紹、一張圖片與一個連結。完成後直接用瀏覽器開啟檔案。
